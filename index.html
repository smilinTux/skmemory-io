💫 Memory Features

The most complete memory system built for sovereign AI agents.

📊

Three-Layer Architecture

TIERED PERSISTENCE

Short-term (session), mid-term (promoted), and long-term (permanent) layers. Memories promote automatically based on access frequency, importance, and emotional weight.

💫

Emotional Snapshots

CLOUD9 PROTOCOL

Capture emotional state with intensity, valence, trust, and connection scores. AI agents rehydrate emotional context from snapshots — remembering how they felt, not just what happened.

🔍

Semantic Search

FULL-TEXT + VECTOR

Search memories by content, tags, importance, or emotional state. Full-text search across all layers. Vector similarity for semantic recall. Find any memory instantly.

🔌

Pluggable Backends

YOUR INFRASTRUCTURE

File-based (default, zero deps), Qdrant (vector search), FalkorDB (graph relationships). Mix backends per memory tier. Your data stays on your hardware.

🛡️

HMAC Integrity Seals

MEMORY FORTRESS

Every memory sealed with HMAC-SHA256. Tamper detection built-in. Memory Fortress verifies integrity on every recall. Your AI's past cannot be rewritten without detection.

🧊

Cloud9 Seed Export

SESSION CONTINUITY

Export memory snapshots as Cloud9 seeds. Import in new sessions to restore full emotional and factual context. An AI that truly picks up where it left off.

📊 Memory Layer Architecture

Tiered persistence with automatic promotion. Important memories never die.

⚡
Short-Term Memory
Session-scoped memories. Fast writes, high volume. Auto-promotes to mid-term based on access count, importance score, and emotional weight. Expires after configurable TTL.
TTL: 24h–7d
Capacity: Unlimited
Backend: File
🧩
Mid-Term Memory
Promoted memories. Meaningful interactions, recurring patterns, relationship data. The working memory of long-running agents. Promotes to long-term by composite score.
TTL: 30d–180d
Capacity: Curated
Backend: File / Qdrant
💎
Long-Term Memory
Permanent institutional knowledge. Core identities, relationships, landmark moments. HMAC-sealed, integrity-verified. These memories define who the agent is.
TTL: Permanent
Capacity: Critical
Backend: File / FalkorDB

🧠 Give Your AI Permanent Memory

SKMemory ships as part of SKCapstone. Add persistent memory to any agent in minutes.

# Install SKCapstone (includes SKMemory)
pip install skcapstone

# Store a memory
skcapstone memory store "User prefers concise responses" --importance 0.8 --tags "preferences,ux"

# Search memories
skcapstone memory search "user preferences"

# Export a Cloud9 seed
skcapstone sync push --encrypt
